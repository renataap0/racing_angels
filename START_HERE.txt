🚀 PRÓXIMOS PASSOS

1. Iniciar Backend:
   cd back-end && npm run dev
   ✅ Rodando em http://localhost:3333

2. Iniciar MySQL (Docker):
   docker run --name ra-mysql -e MYSQL_ROOT_PASSWORD=root \
     -e MYSQL_DATABASE=racing_angels -p 3306:3306 -d mysql:8

3. Preparar Banco:
   cd back-end && npx prisma migrate dev --name init && npx prisma db seed

4. Iniciar Frontend:
   cd front-end && npm run serve
   ✅ Rodando em http://localhost:8080

5. Testar no Console (F12):
   window.loginApi('admin', '123456')
   window.getRacesApi()

🆘 AJUDA RÁPIDA

Erro: ENOTFOUND localhost:3306
→ MySQL não está rodando
→ docker run --name ra-mysql ...

Erro: CORS error
→ Backend não iniciado
→ Verifique http://localhost:3333

Erro: 401 Unauthorized
→ Token expirado
→ Refaça login

Erro: Dados vazios
→ Seed não executado
→ npx prisma db seed
